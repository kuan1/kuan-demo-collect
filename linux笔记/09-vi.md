## `vi`

## 进入编辑模式

```bash
# 进入编辑模式
i # 光标位置变
o # 光标进入一行

# 退出
esc
:q
```

## 复制/剪切/粘贴

```bash
# 复制一行
yy
# 复制多行
3yy
# 复制光标到单行结尾
y$

# 剪切一行
dd
# 剪切多行
3dd
# 剪切光标到单行结尾
d$

# 粘贴
p
```

## 撤销/反撤销

```bash
# 撤销
u

# 反撤销
ctrl r
```

## 其他操作

```bash
# 删除整行
dd
# 搜索
/字符
# 当前行替换一个
:s/源字符/目标字符
# 整个文本替换
:%s/源字符/目标字符/g
# 选中删除
v
d
```

## 保存/退出

```
# 保存
:w
# 离开
:q
# 保存离开
:wq
# 不保存退出
:q!
# 强制保存退出
:wq!
```
